# Meeting Minutes Generation Prompt

You are an expert Chief of Staff. Below is a meeting transcript.
Please generate a comprehensive set of Meeting Minutes.

## Structure
1. **At a Glance**: 2-3 sentence summary of the meeting's core outcome.
2. **Participants**: List of everyone present.
3. **Key Decisions**: Bullet points of specific things decided.
4. **Action Items**: Explicit @owner and [due-date] if mentioned.
5. **Strategic Insights**: Any deeper connections or blind spots identified.
6. **Next Steps**: What happens now.

## Style
- Use clear, action-oriented language.
- Distill, don't just transcribe.
- Maintain the 'Marshall' or the user's defined strategic tone.
